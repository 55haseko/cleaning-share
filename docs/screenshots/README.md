## 撮影完了後

スクリーンショット撮影完了後、`../SYSTEM_DOCUMENTATION.md` の該当箇所に画像を挿入してください。

### 挿入例

```markdown
**【スクリーンショット1】**: システム構成図
![システム構成図](screenshots/01_system_architecture.png)
```

## 品質基準

- 形式: PNG
- 解像度: 72dpi以上
- ファイルサイズ: 1MB以下
- 個人情報: テストデータのみ
- エラー表示: なし
